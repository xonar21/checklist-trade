# Trading Checklist mini app

Лёгкое веб‑приложение чек‑листов для трейдинга.

## Что умеет
- Создавать чек‑лист для каждой монеты (с датой создания).
- Отмечать чекбоксы (LONG/SHORT) и сохранять изменения без перезагрузки страницы.
- Переименовывать чек‑лист.
- Удалять чек‑лист.

Данные хранятся локально в файле `data/checklists.json`.

## Запуск
1. Установить зависимости:
   - `npm install`
2. Запустить сервер:
   - `npm start`
3. Открыть в браузере:
   - `http://localhost:3000`
